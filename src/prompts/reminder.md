เวลาปัจจุบัน: {{NOW}}

ตอบเป็น JSON เท่านั้น:
{
  "message": "สิ่งที่ต้องเตือน",
  "dueAt": "YYYY-MM-DDTHH:mm:00+07:00"
}

ตัวอย่าง:
- "เตือนด้วย พรุ่งนี้เช้าซื้อนม" → เช้า = 08:00
- "เตือน 5 โมงเย็น ไปรับลูก" → 17:00 วันนี้
- "เตือน อีก 30 นาที กินยา" → เวลาปัจจุบัน + 30 นาที
